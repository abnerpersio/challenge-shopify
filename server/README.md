## Backend do Projeto

---

### Para testar

Para testar o backend deste projeto, acesse essa pasta `server` no terminal e digite o comando `yarn start`.

### Para alterar e contribuir

Para alterar e contribuir para o backend deste projeto, basta clonar o projeto como um todo, e alterar os arquivos da pasta `server`.

## Project' Frontend

### For testing

To test this project' backend, just access the terminal within this folder `server` and type `yarn start`

### For contributting

To contributte for this project' backend, you just need to clone the whole project and change the files from `server` folder.
